SaveTogether Video Integration Guide
===================================

To add a video to the project:

1. Place your video file in this directory (static/videos/)
2. Supported formats: MP4, WebM, Ogg
3. Recommended format: MP4 with H.264 codec
4. Recommended size: 1920x1080 or 1280x720
5. Rename your video file to "savings-intro.mp4" to match the template

Example:
- static/videos/savings-intro.mp4

To add a poster image:
1. Place your poster image in static/images/
2. Name it "video-poster.jpg"
3. Recommended size: 1920x1080 or 1280x720

The video will automatically appear on the homepage.
